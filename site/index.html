Catchphrase!

An iOS app for EECS 493
Nick Gerard, Braden Schmidt, Scott Karl, & Kurt Jensen

The Game

We have created a mobile version of the popular party game Catchphrase.

Catchphrase is usually played in groups (4+ people), on two teams, and the objective is to get your team to guess the word on the screen (by saying clues that do not include the actual word or rhyme with it). When the word is guessed correctly, the device is passed to the next person, and when the timer runs out, the team that is not holding the device wins the round.

The App

Built for iOS. Optimized for any size iPhone, iPod touch, and iPad, and can be used in both landscape and portrait.

Our app allows for the creation of different teams and games, with customizable settings. The actual gameplay is an intuitive interface of the word and timer (optionally visible), and the next word is shown with a simple swipe gesture and animation. When the round is near completion, the device vibrates, and a "time's up" sound plays when it is over and the user is prompted to record who won.

Game Lobby

The game lobby grid allows the user to view ongoing games or create a new game.

Teams

The team grid allows the user to create a new game or view performance history and team stats.

Settings

The settings view allows the user to change the word library, customize the timer, set the number of rounds, delete all data, and view game rules.

Gameplay

The game view allows the user to swipe to change words. Sounds/Vibrations signal the end of the round and a menu appears for easy scoring.
